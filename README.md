🎬 Netflix-Inspired Movie Management System

📌 Project Overview

🎥 A full-stack web application inspired by Netflix for managing movie data

💻 Built using modern web technologies with a focus on UI/UX and performance

🔄 Supports complete CRUD operations (Create, Read, Update, Delete)

🛠️ Tech Stack

⚛️ Frontend: React.js (Vite)

🎨 Styling: Custom CSS (No frameworks used)

🌐 Backend: Node.js + Express.js

🗄️ Database: MongoDB (Mongoose)

🎨 User Interface Features

🌑 Dark theme UI (#141414 background)

🔴 Red accent styling (#E50914 like Netflix)

📱 Fully responsive design (mobile + desktop)

🎞️ Horizontal scrolling movie rows (Netflix-style)

🎥 Hero banner with featured movie

🧭 Navbar with search and navigation

⚙️ Core Functionalities (CRUD)

➕ Add Movie

Form to enter movie details (title, genre, rating, etc.)

👀 View Movies

Display movies in attractive card layout

✏️ Edit Movie

Update existing movie information

❌ Delete Movie

Remove movies with confirmation popup

🎬 Movie Display Features

🖼️ Poster-based movie cards

✨ Hover animations (scale + details reveal)

📂 Categorized rows (Trending, Top Rated, Action)

🔍 Search movies by title

🎭 Filter movies by genre

🔌 Backend & API

📡 RESTful API integration

📥 Fetch movies from database

📤 Send data from frontend to backend

🔄 Real-time updates without page reload

🧠 Database Design

🗃️ Stores movie details like:

Title

Description

Genre

Release Year

Rating

Image URL

✨ Extra Features

🔔 Custom toast notifications

⏳ Loading spinner

📦 Clean and structured project architecture

🎯 Smooth animations using CSS

🎯 Project Outcome

🚀 Fully functional full-stack application

🎬 Real-world Netflix-like UI experience

💡 Demonstrates frontend + backend integration

🧑‍💻 Useful for learning CRUD operations and UI design
